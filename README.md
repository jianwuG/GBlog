### GBlog项目
* 前台 blog
    * 基于nextjs+hooks+react-redux+immutable+react-redux
* 后台 admin
   * 基于hooks+react-redux+immutable+react-redux+redux-immutable

* 中台 server
    * 基于egg+egg-mysql+egg+egg-cors
